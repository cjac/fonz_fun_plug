SQUEEZECENTER FUN-PLUG

Download fun_plug and fun_plug.tgz to Volume_1, then reboot the NAS device
using the Web interface (Tools -> System -> Restart). 

Note: Do not use the web-based download tool!

After reboot, the fun-plug will automatically install the Squeezecenter package.
After a few minutes, the Squezecenter Web interface becomes available at 
  http://xxx:9000/ 
(replace xxx with the name or IP address of your NAS device).
